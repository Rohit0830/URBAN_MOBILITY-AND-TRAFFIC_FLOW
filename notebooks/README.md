# Notebooks

This directory contains Jupyter notebooks intended as a **self-learning** playground for basic data analysis and EDA. Keep notebooks short, explanatory, and output-stripped (use `nbstripout` or pre-commit hooks). Prefer `dataset/sampled_probe.csv` for quick experiments; add small, well-documented example notebooks rather than large, heavy computations.
